# Projetofinal

Grupo: Tallison Miranda, Luiz Dias, Athur Nunes, Vinícius Angelo, Eduardo Nunes.

O nosso projeto sera um site para organizar e planejar campanhas de D&D, com duas ou três páginas, uma homepage com informações e regras sobre a campanha e uma página contendo o cadastro dos jogadores e seus personagens. Uma página admin para cadastrar jogadores e personagens.
